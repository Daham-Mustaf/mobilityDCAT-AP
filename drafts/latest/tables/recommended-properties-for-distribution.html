+data format version<

mobilitydcatap:dataFormatVersion

rdfs:Literal

This property describes the version of the Grammar, Schema, or Data Model of the delivered content within the Distribution (see the corresponding properties). A version might be, e.g., DATEX II v3.2. This information should be provided in a textual format.

0..1

+data model schema

mobilitydcatap:dataModelSchema

skos:Concept

This property describes the schema of the delivered content, as applied for the data model within the Distribution (see property "data model"). The schema can be individually determined by the data provider (e.g., a stakeholder-based DATEX II profile) or a prescribed by other institutions (e.g., a DATEX II recommended reference profile). The data platform should keep references of commonly used schemas.
