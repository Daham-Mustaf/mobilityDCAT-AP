Class name Distribution
Obligation Mandatory
URI

dcat:Distribution

Usage note

A physical embodiment of the dataset in a particular format. Thiy class has been classified in DCAT-AP as ‘Recommended’, to allow for cases that a particular Dataset does not have a downloadable Distribution. However, for the context of mobility data portals, it can be expected that all datasets have downloadable Distributions. The provision of class Distribution is raised to mandatory.
