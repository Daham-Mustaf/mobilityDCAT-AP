dct:accrualPeriodicity

dcat:Dataset

EU Authority Table "Frequency" [[EUV-FREQ]]

http://publications.europa.eu/resource/authority/frequency

This is a mandatory vocabulary "Frequency" by the Publications Office of the European Union.

The values from this vocabulary do not cover all relevant time intervals used in mobility data portals, e.g., minute-based intervals. Thus, there is a second vocabulary created by mobilityDCAT-AP (see next row), adding any missing time intervals.

+mobilityDCAT-AP Vocabulary "Update frequency"

+Placeholder for link to vocab "Update frequency"

This is an vocabulary created by mobilityDCAT-AP, complementing time intervals missing in the EU vocabulary (see row above).

dct:format

dcat:Distribution

EU Authority Table "File Type" [[EUV-FT]]

http://publications.europa.eu/resource/authority/file-type

This is a mandatory vocabulary "File Type" by the Publications Office of the European Union.

Please note that mobilityDCAT-AP has different properties related to the format, including mobilitydcatap:dataModel, dct:format, cnt:characterEncoding, mobilitydcatap:grammar, and mobilitydcatap:dataModelSchema. This controlled vocabulary relates to dct:format, i.e., the base standard that specifies syntactically correct documents.
